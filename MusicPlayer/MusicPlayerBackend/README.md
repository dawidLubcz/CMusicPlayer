# MusicPlayerCore
For fun project to implement simple and small mp3 player for raspberrypi (or other linux based system).

## How to build
```
mkdir build
cd build
bash ../install.sh
cmake ..
cmake --build ..
```

## Usage
```
./MusicPlayerCore
```
